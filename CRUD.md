# CRUD
- Create, Read, Update, Delete의 줄임말.
- 데이터베이스나 API같은 곳에서 쓰이는 기본적인 함수, 액선, 패러다임 이라고 생각하면 될듯 함
- Create: 새로운 정보를 만들어서 추가
- Read: 기존에 있던 정보를 가져오기
- Update: 기존에 있던 정보를 편집
- Delete: 정보를 삭제함. 
- SQL에서...:
	- INSERT, SELECT, UPDATE, DELETE가 똑같은 역할을 함.
